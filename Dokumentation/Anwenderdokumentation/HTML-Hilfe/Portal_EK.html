Einsatz und Kräfte

Das Modul "Einsatzkräfte" beitet die Möglichkeit Kräfte, Einsatzschwerpunkte sowie Module und Ortsverbände anzulegen und zu verwalten. Als Kräfte zählen Helfer, Kraftfahrzeuge (KFZ) und Einheiten.

Der Einsatzmanager



Der Einsatzmanager

Im Einsatzmanager können Einsatzschwerpunkte, Module, Einheiten und Helfer eingesehen werden. Es besteht die Möglichkeit Helfer oder KFZs per Drag & Drop zwischen Einheiten zu verschieben. Einheiten per Drag & Drop Modulen zu zuordnen und Module an Einsatzschwerpunkte zu verlegen. Dazu fasst man den Helfer (analog für alle anderen) an und zieht ihn auf die Einheit zu der er zugeordnet werden soll. Per Doppelklick auf ein beliebiges Element wird dieser in dem ensprechenden Reiter angezeigt.
Hinweis Das Doppelklicken funktioniert nicht mit Materiallien, diese werden über das Portal Logistik verwaltet. Der baumstrukturierte Einsatzmanager bietet zahlreiche Möglichkeiten.
So konnen z.B. die Knoten durch Kontextmenüs oder Doppelklick bedient werden. Es stehen die folgendne Optionen zur Auswahl:

Helfer

Helfer als verpflegt kennzeichenen
Speichert die Aktuelle Urzeit als Verpflegungszeit des Helfers

Helfer anlegen
Erstellt einen neues Helfer "Helfer Antonon"

Daten bearbeiten (Doppelklick)
Öffnet die Einstellungen des Helfers im entsprechenden Reiter

Einheiten/Module

Neues Modul/Neue Einheit
Erstellt ein neues leeres Modul/eine leere Einehit dem Personen, Fahrzeuge und Geräte zugeordnet werden können.

Verpflegung Erfolgt
Speichert die Aktuelle Urzeit als Verpflegungszeit der HElfer die aktueller der Einheit zugeordnet sind

Konfiguration bearbeiten (Doppelklick)
Öffnet die Einstellungen der Einheit in der linken Programmhälfe

Einsatzschwerpunkt

Erstellen
Erstellt einen neuen Einsatzschwerpunkt "neuer Einsatzschwerpunkt". Dies Kann als Container für Einheitn/Module dienen.

Bearbeiten (Doppelklick)
Öffnet die Angaben zum Einsatzschwerpunkt im entsprechenden Reiter

Der Reiter "Helfer"



Helfer

Hier können die Detaildaten des Helfer konfiguriert werden. Beim Klicken auf "Speichern" werden diese an den Server übermittelt.
Beim Klicken auf Zurücksetzen werden diese verworfen, und durch das erneute Füllen der Felder kann ein Neuer Helfer angelegt werden. So ist ein zügiges Erstellen von Helfern möglich.
Durch Klicken auf den jeweiligen Eintrag im Grid wird der angecklickte Helfer geladen, und kann editiert, und gespeichert werden.

Der Reiter "Einheit"



Einheit

Hier können die Detaildaten der Einheit konfiguriert werden. Beim Klicken auf "Speichern" werden diese an den Server übermittelt.
Beim Klicken auf Zurücksetzen werden diese verworfen, und durch das erneute Füllen der Felder kann ein Neuer Helfer angelegt werden. So ist ein zügiges Erstellen von Einheiten möglich.
Führer und stellv. Führer einer Einheit dürfen nicht identisch sein.

Der Reiter "Einsatzschwerpunkte"



Einsatzschwerpunkte

Hier können die Detaildaten des Einsatzschwerpunktes konfiguriert werden. Beim Klicken auf "Speichern" werden diese an den Server übermittelt.
Beim Klicken auf Zurücksetzen werden diese verworfen. Wird in der Auswahlliste "Neuen Einsatzschwerpunkt anlegen" ausgwählt, so kann ein Neuer Einsatzschwerpunkt mit Daten gefüllt und anschließend gespeichtert werden. So ist ein zügiges Erstellen von Einaptzschwerpunktern möglich.

Der Reiter "Modul"



Modul

Ein Modul als Container für Einheiten zeichnet sich nur durch seinen Namen aus. Dieser kann hier erfasst und gespeichert werden.

Der Reiter "Ortsverbände"



Ortsverbände

Nach dem Klicken auf "Ortsverbände" kann nach de klicken auf "Zurücksetzen" ein neuer Ortsverband eingegeben werden, oder durch klicken auf einen Eintrag im Datagrid ein bestehender Ortsverband verändert werden.
