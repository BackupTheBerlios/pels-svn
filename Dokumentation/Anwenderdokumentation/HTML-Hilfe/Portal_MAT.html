Aufträge


Dieses Portal erlaubt Meldungen, Aufträge & Termine in das System einzupflegen.

Meldung abfassen



Meldung abfassen

Bei Meldung abfassen können eingehende Meldungen ins System eingegeben werden. Dabei ist der aktuelle Benutzer der Autor. Die Meldung kann an alle Kräfte gehen, die aktuell im System erfasst sind. Zusätzlich kann noch ein interner Empfänger angegeben werden. Der interne Empfänger ist Benutzer im System. Die Meldung wird in dessen ToDo-Liste angezeigt, bis sie zur Kenntnis genommen wurde.
Neben Meldungen können auch Erkundungsergebnisse abgefasst werden. Sie unterscheiden sich im Vergleich zu Meldungen dadurch, dass Sie mehr geordnete Daten aufnehmen.


Eingabemaske Erkundungsergebnis

Zusätzlich ist es möglich einen Kommentar (ganz unten) zu einem Erkundungsergebnis zu erfassen.
Durch Drücken auf den Button Speichern wird die Meldung gespeichert. Zurücksetzen verwirft die aktuellen Eingaben. Speichern & Drucken speichert die Meldung und wechselt dann in das Portal Report zur Druckvorlage der aktuellen Meldung.

Meldung erfassen


unterscheidet sich von dem oben beschriebenen Meldung abfassen dadurch, dass der Autor nicht der zwangsläufig der aktuelle Benutzer ist. So können also auch Meldungen & Erkundungsergebnisse von Dritten (nicht als Benutzer im System) erfassten Personen aufgenommen werden.
Hinweis: Nicht alle Rollen im System haben das Recht Meldungen Dritter zu erfassen.

Auftrag eingeben


Hier können Aufträge eingegeben werden. Neben dem eigentlichen Auftragstext und den aus den anderen Reitern bekannten Informationen kann auch eine Befehlsart bestimmt werden, sollte es sich nicht um einen normalen Auftrag.


Aufträge eingeben

Sollten der Benutzer Aufträge nachverfolgen wollen, so kann er dass durch Markieren der Checkbox veranlassen.

Aufträge nachverfolgen


Aufträge nachverfolgen

Alle Aufträge, die zur Nachverfolgung gekennzeichnet wurden erscheinen in diesem Reiter als Tabelle. Diese können eingesehen und ausgewählt werden.
Ausgewählte Aufträge können durch Drücken auf den Button Speichern als ausgeführt gekennzeichnet werden. Sie werden danach nicht mehr nachverfolgt und erscheinen auch nicht mehr in der Tabelle. Dabei wird auch der Ausführungszeitpunkt gesetzt.

Termin erstellen


Termin erstellen

Hier können neue Termine für Nutzer erstellt werden. Durch Drücken auf Zurücksetzen werden die Eingaben verworfen, durch Drücken auf Speichern wird der Termin gespeichert.
Gespeicherte Termine werdem beim angegebenen Empfänger in der ToDo-Liste angezeigt (falls das Flag gesetzt wurde),bis Sie zur Kenntnis genommen wurden.

Hinweis: Die Optionen 'ist wichtig' und 'Erinnern am' finden in der aktuellen Version noch keine Berücksichtigung.
